Python 1 Final Project - Emu Brawl

Developed in Highschool Python 1 course, which goes over basic Python programming development. This is the first major project I ever developed, and used what tools I had available from the class to create a game which I titled "Emu Brawl". Emu Brawl is a text based "adventure" game which involves using a rock paper sissors system (turned into hit, shoot, and deflect) to fight enemies with specific skillsets. These enemies have different probabilities of picking certain options over another, like a kangaroo is more likely to hit then deflect, and cannot shoot at all. The game itself went way over the expectations of the assignment, and became more of a passion project for me to explore how far my Python skills could go at that current point. When looking back on it, there are many ways this program can be optimized, which demonstrates a good growth in my programming knowledge and how far I've improved since I began. Project link: https://github.com/IRStotts/Python-1-Sem-Project.git

Personal Discord Bot - Bobby Blooper

Bobby Blooper is a passion project discord bot developed in October 2024. It is the successor to my previous bot, Beanos-Bot, which was named after my first discord handle and lost after a file transfer error. Bobby Blooper is able to perform a large list of tasks, which so far consists of:

  • A basic greeting to the user
  • Tell a joke
  • Coin flip
  • 8 ball responses
  • Random fortune cookie slip
  • Random number generator
  • Match % of 2 users
    • This % will be assigned for the two users until the bot restarts
  • Play rock paper siccors
  • Play would you rather
  • Assign a One Piece character to a user
    • This character will be assigned to the user until the bot restarts
  • Generate a random One Piece bounty
  • A mini-dungeon campaign
    • A random text adventure, consisting of 20 dungeon floors and a final Cyclops boss
    • Events are pulled from a random list and has a corresponding randomized outcome
    • Health, items, gold, and inventory are all managed for each user
    • Progress will remain assigned until the bot restarts
This project is currently a work in progress and is still being updated with new features. Eventually, a more polished version will be published to Github as development continues.
